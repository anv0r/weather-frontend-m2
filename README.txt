# clima world-frontend-m2

## Proposito
MVP de una aplicación de clima desarrollado para el módulo 2, aplicando HTML5 semántico, Bootstrap 5 y JavaScript básico para la manipulación del DOM y el manejo de estados simples.

## Funcionalidades
1.  Home:Listado de 10 localidades en un grid responsivo.
2.  Detalle:Al hacer clic, se muestra la vista con datos detallados y el pronóstico de 7 días (generado dinámicamente).
3.  Navegación:Navbar con estado activo dinámico.

## Tecnologías
* HTML5 Semántico
* Bootstrap 5.3
* JavaScript
